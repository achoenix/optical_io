## Introduction


### Pros and Cons

**Pros**
- Lower power consumption
- Improved network flexibility and system availability
- Lower latency relative to packet switches
- Data rate / signal type agnostic


**Cons**
- Deploying OCS needs a new control plane
- Different physical deployment requirements than packet switches
- Currently vendor-specific (APIs, management tools, physical architectures)
